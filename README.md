Hey! 👋

DevVault is a simple project I built to keep all my developer resources in one place. Instead of having dozens of random bookmarks, this app lets me organize useful websites by title and category, search through them, and quickly open them whenever I need them.

This project was also a great way for me to practice Next.js, React, and working with Local Storage.

Features
Add a new resource
Search resources by title
Filter by category
Delete resources
Open resources in a new tab
Saves everything using Local Storage, so your data stays even after refreshing
Built With
Next.js
React
JavaScript
CSS
Getting Started

Clone the repository:

git clone <your-repository-link>

Install dependencies:

npm install

Run the development server:

npm run dev

Then open:

http://localhost:3000
